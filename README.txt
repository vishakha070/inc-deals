This plugin creates a CPT that lists all deals and creates shortcode for display all deals and for filters.

== Description ==

This plugin helps to add deal as cpt, provides a shortcode for display all deals with filters and a shortcode for display single deal.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `inc-deals.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
